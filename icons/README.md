BilMart is a secondhand marketplace designed for Bilkent University students. The platform allows students to list, and purchase used products within categorized sections such as books, clothing, electronics. 
The main goal of this project is to provide an efficient and student-exclusive marketplace that supports the reuse of academic materials. 
BilMart is implemented using Java Swing for the graphical user interface (GUI) and Java for backend development. The system will integrate with a remote MySQL database to store user data, product listings, and ratings.
To run the code, the jar files in the lib folder should be added to the referenced libraries.
